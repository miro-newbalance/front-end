📝 Практическое применение регулярных выражений

1. Метод RegExp.test()

Проверяет, соответствует ли строка регулярному выражению

Пример: Проверка email

Пример: Проверка даты (формат: ДД.ММ.ГГГГ)

2. Метод RegExp.exec()

Ищет совпадение в строке и возвращает результат

Пример: Поиск всех чисел в тексте

Пример: Извлечение данных

3. Методы String

String.match()

Находит все совпадения с регулярным выражением

String.replace()

Заменяет совпадения в строке

4. Флаги регулярных выражений

g - глобальный поиск
i - игнорирование регистра
m - многострочный режим
s - точка включает перенос строки
u - поддержка Unicode

Пример с флагом i (игнорирование регистра)

Пример с флагом g (глобальный поиск)

5. String.split() с регулярными выражениями

Пример: Разделение по нескольким разделителям

String.search()

Возвращает индекс первого совпадения
